# NutriTrack Server Setup Instructions

Follow these steps to deploy NutriTrack on your home server (behind Cloudflare Tunnel).

## 1. Prerequisites
Ensure you have Python 3.10+ installed on your server.

## 2. Clone the Repository
```bash
git clone https://github.com/looser-xxx/nutriTracker.git
cd nutriTracker
```

## 3. Set Up Virtual Environment
It is highly recommended to use a virtual environment.
```bash
python3 -m venv venv
source venv/bin/activate  # On Linux/macOS
```

## 4. Install Dependencies
```bash
pip install -r requirements.txt
```

## 5. Configure Environment Variables
Create a `.env` file in the root directory:
```bash
touch .env
```
Open `.env` and add your Google OAuth credentials and a secret key:
```env
GOOGLE_CLIENT_ID=your_id_here.apps.googleusercontent.com
GOOGLE_CLIENT_SECRET=your_secret_here
SECRET_KEY=generate_a_random_secret_key_here
```

## 6. Initialize the Database
Run the initialization script to create the SQLite database and tables:
```bash
python3 init_db.py
```

## 7. Cloudflare Tunnel Configuration
- Point your Cloudflare Tunnel to `http://localhost:5000`.
- Ensure your Google Cloud Console "Authorized redirect URIs" includes:
  `https://nut.realitystudios.in/auth/callback`

## 8. Run the Application
For development/testing:
```bash
python3 app.py
```
For production (using Gunicorn):
```bash
pip install gunicorn
gunicorn -w 4 -b 0.0.0.0:5000 "app:createApp()"
```

## 9. Security Note
The `.env` file and the `instance/` folder are excluded from Git to protect your secrets and data. Always keep your `.env` file secure on the server.
